# Blog Web Application

## Overview

This Blog Web Application is a Django-based platform that allows users to create, read, update, and delete blog posts. It is designed with simplicity and usability in mind, making content creation and management straightforward.

## Features

- Create and publish blog posts
- View posts in list and detailed formats
- Edit existing blog content
- Delete outdated or unwanted posts
- Clean and responsive user interface

## Tech Stack

- **Backend**: Django (Python)
- **Frontend**: HTML, CSS, Bootstrap
- **Database**: SQLite (default with Django)

## Requirements

- Python 3.x
- Django 3.x or higher

## Setup and Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yitbarek16/My-Blog.git

2. **Create and activate a virtual environment**:
     ```bash
     python -m venv venv
     source venv/bin/activate

3. **Run database migrations**:
     ```bash
     python manage.py migrate

4. **Start the development server**:
     ```bash
     python manage.py runserver
     
5. **Access the application**:
     Open your web browser and navigate to http://127.0.0.1:8000/.
